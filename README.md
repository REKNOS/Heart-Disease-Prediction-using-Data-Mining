# Heart-Disease-Prediction-using-Data-Mining
This project applies machine learning techniques to predict the likelihood of heart disease using clinical patient data. By analyzing features such as age, cholesterol levels, blood pressure, and heart rate, the goal is to identify key risk factors and support early detection strategies in healthcare.
The project follows a complete data mining pipeline, including data preprocessing, exploratory data analysis (EDA), model development, and performance evaluation. Multiple classification models were implemented and compared to determine the most effective approach for predicting heart disease.

❓ Research Question

Can we accurately predict the presence of heart disease using routine clinical data?
Additionally, which features contribute the most to identifying high-risk patients?

⚙️ Methods Used
Data Cleaning and Preprocessing
Exploratory Data Analysis (EDA)
Machine Learning Models:
Logistic Regression
Decision Tree
Random Forest
Model Evaluation:
Accuracy, Precision, Recall, F1-Score
ROC-AUC
Cross-Validation
📈 Key Results
Logistic Regression and Random Forest achieved the highest accuracy (~83%)
Random Forest showed the highest precision, reducing false positives
Logistic Regression provided the best balance between interpretability and performance
ROC-AUC scores (~0.94–0.95) indicate strong predictive capability
🧠 Key Insights
Heart disease can be effectively predicted using routine clinical features
Chest pain type, maximum heart rate, and thalassemia are strong predictors
Machine learning models can assist healthcare professionals in early diagnosis and decision-making
